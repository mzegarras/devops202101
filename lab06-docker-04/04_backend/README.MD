# PROGRAM REST SERVICE
## Getting Started
## Prerequisites
**Software**
* NodeJS v13.12.x or higher
* Npm v6.14.x or higher
* Docker 17.12.x or higher
* Docker Compose: 1.18.x or higher

## Building

**Container**

```
docker-compose build
```
## Running

Run dependencies:

```
docker-compose run
```
